# [nvm](https://github.com/nvm-sh/nvm)

nvm, for Node Version Manager, is a POSIX-compliant bash script to manage multiple active node.js
versions.

**Install:**

<details>
<summary>MacOS</summary>

Use the command recommanded in the
[install section](https://github.com/nvm-sh/nvm?tab=readme-ov-file#installing-and-updating) without
updating shell configuration files (since the configuration is already in this repository).

```bash
PROFILE=/dev/null bash -c 'curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash'
```

</details>
